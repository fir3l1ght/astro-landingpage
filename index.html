Características

Astro Islands

Componente de UI interactivo en una página HTML predominantemente estática.

MPA

Una Aplicación de múltiples páginas (MPA) es un sitio web que consta de varias páginas HTML, en su mayoría renderizadas en el servidor.

Integraciones

Las Integraciones de Astro agregan nuevas funcionalidades a su proyecto con solo unas pocas líneas de código y permiten usar otros frameworks y herramientas como Vue o Tailwind.

Soporte de Markdown Nativo

Astro permite el uso de Markdown ya que trata cualquier archivo .md (u otra extensión alternativa soportada) o .mdx dentro de la carpeta /src/pages como una página.

Enfocado en el Contenido

Astro fue diseñado para sitios web ricos en contenido.

Servidor Primero

Los sitios web se ejecutan más rápido cuando procesan HTML en el lado del servidor.
